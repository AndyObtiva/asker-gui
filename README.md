# ASKER GUI

`asker-gui` gem, graphical front end for `asker-tool`gem.

# Description

* Free Software [LICENSE](LICENSE).
* Multiplatform.
* Ruby program.

# Installation

1. Install Ruby on your system.
2. `gem install asker-gui`

# Usage

1. Open a terminal.
2. Run `asker-gui` command.
